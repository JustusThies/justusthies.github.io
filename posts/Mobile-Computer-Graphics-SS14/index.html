Lecture: Mobile Computer Graphics

Summer Term 2014
2014, Apr 10    

Lecturer

Marc Stamminger

Teaching Assistants

Christoph Weber Justus Thies

Content

The lecture deals with computer graphics algorithms for mobile devices. Topics include special algorithms for the limited resources of mobile devices, as well as special applications of mobile computer graphics platforms. The following topics are covered in detail:

  • Hardware and rendering architectures of heterogeneous mobile devices
  • Special graphic algorithms for mobile devices
  • Compression method for textures and geometry
  • Mobile VR output devices
  • Algorithms and Devices for Augmented Reality

Prerequisites

C++, Java
